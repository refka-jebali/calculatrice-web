# Calculatrice Web

Projet simple de calculatrice web réalisé dans le cadre du TP Jenkins.

## Description
Cette application est une calculatrice basique qui permet d’effectuer les opérations suivantes :
- Addition
- Soustraction
- Multiplication
- Division

## Technologies utilisées
- HTML
- CSS
- JavaScript
- Jenkins (CI/CD)
